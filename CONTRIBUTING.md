# 🤝 Como Contribuir

## Para Pesquisadores

1. Fork o repositório
2. Adicione dados em `dados/[esfera]/nome-sobrenome.md`
3. **OBRIGATÓRIO**: inclua link para fonte oficial (.gov.br, .jus.br)
4. Abra Pull Request

## Para Desenvolvedores

1. Melhore os scripts em `scripts/`
2. Adicione novos coletores para APIs públicas
3. Testes são bem-vindos

## Regras Absolutas

- Todo dado DEVE ter fonte oficial linkada
- Sem opinião, viés ou linguagem pejorativa
- Presunção de inocência (Art. 5º CF)
- Neutralidade jornalística
- PRs passam por revisão de fontes antes do merge
